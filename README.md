# Github Search

Herramienta para buscar usuarios en Github

## Como instalar localmente

Clone the project

```bash
  git clone https://github.com/NicolasCasClew/Github-Search-Project.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  ng serve --open
```
